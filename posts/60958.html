HTML全局属性

  • accesskey 规定激活元素的快捷键（使元素获得焦点）

  • class 规定元素类名

  • contenteditable 规定元素内容是否可编辑

  • data-* 嵌入自定义数据

  • dir 规定元素内容的文本方向

  • draggable 规定元素是否可拖动

  • hidden 规定元素是否可见

  • id 规定元素的唯一id

  • lang 规定元素的语言

  • spellcheck 规定是否对元素进行语法检测

  • style 规定元素的CSS行内样式

  • tabindex 规定元素的tab键次序

  • titile 规定关于元素的额外信息

  • translate 规定是否应该翻译元素内容

常用标签

标题

ps

  • 默认情况下，HTML会自动地在块级元素前后添加一个额外的空行——比如段落，标题元素前后
  • 确保标题标签只用于标题，搜索引擎使用标题为网页结构和内容编制索引

水平线

hr

段落

p

在段落中折行使用 br

链接

a

属性：

  • href 规定链接的目标

  • target 规定被链接的文档在何处显示（当前页面或者一个新页面）

  • name 规定锚的名称，可以使用name属性创建HTML页面中的书签

图像

img

属性：

src 图像的URL地址

Alt 为图像定义一串预备的可替换文本

map

定义一个图像映射（image-map）带有可点击区域的一副图像

area

定义图像映射中的区域

引用和术语定义

  • abbr 定义缩写 使用全局属性 title

  • acronym 定义首字母缩写

  • adderss 定义文章作者的联系信息，其中的文本通常为写题，大多数浏览器会在元素前后加上换行

  • blockquote 标记长引用

    blockquote 与 /blockquote 之间的所有文本都会从常规文本中分离出来，经常会在左、右两边进行缩进（增加外边距），而且有时会使用斜体。也就是说，块引用拥有它们自己的空间。

  • q 定义短引用，浏览器通常在引用的内容周围添加引号 本质上与 blockquote相同， 不同的是他们的显示和应用

  • cite 定义引用，引证，它可以使你或者其他人从文档中自动摘录参考书目，定义著作的标题

  • dfn 等一一个定义项目
