事物

简介

一组操作的集合，是一个不可分割的工作单位，事物会把所有的操作作为一个整体向系统提交或撤销操作请求，这些操作要么同时成功，要么同时失败

例如转账：张三给李四转账1000块钱，张三银行账户的钱减少1000，而李四银行账户的钱要增加 1000。 这一组操作就必须在一个事务的范围内，要么都成功，要么都失败。

MySQL的事物是自动提交的，当执行完一条DML语句的时候，MySQL会立即隐式的提交事物

  1. 设置事物的提交方式（取消自动提交）

  2. 提交事物

  3. 回滚事物

  4. 开启事物

转账案例

特性

  • 原子性（Atomicity）:是不可分割的最小单元，要么全部成功，要么全部失败
  • 一致性（Consistency）:事物完成时，必须使所有的数据都保持一致状态
  • 隔离性（Isolation）:数据库提供的隔离机制，保证事物在不受外部并发操作影响的独立环境下运行
  • 持久性（Durability）:事物一旦提交或回滚，它对数据库中数据的改变就是永久的

简称ACID

检索语句

语法：select “something” form “somewhere”

示例：

  1. 检索单个列

    select 字段 from 表单

  2. 检索多个列

    select 字段, 字段, 字段 form 表单

  3. 检索所有列

    select * from 表单

  4. 检索不同的行（去重）

    select distinct 字段 from 表单

  5. 限制结果

    select 字段 from 表单 limit 多少行

    select 字段 from 表单 limit 从哪开始（初始下标为 0） 多少行

  6. 完全限定表名

    select 表名.字段 from 表单

排序检索

  1. 按某一字段进行排序

    select 字段 from 表单 order by 字段

  2. 按多个字段进行排序

    select 字段 from 表单 order by 字段1, 字段2

    先按字段 1 排，有多个字段 1 ，再按字段 2 排

  3. 指定排序方向

    select 字段 from 表单 order by 字段1 （des || asc(默认)）, 字段2（des || asc(默认)）

过滤数据

  1. where 子句：根据 where 子句中指定的搜索条件进行过滤，where 子句在 from 子句之后给出

    select 字段 from 表单 where 字段 = 某字段值

  2. where 子句操作符

  3. 检查单个值

    select 字段 from 表单 where 字段 = 某字段值

    select 字段 from 表单 where 字段 < 某字段值

    select 字段 from 表单 where 字段 <= 某字段值

    select 字段 from 表单 where 字段 <> 某字段值

  4. 范围值检查

    select 字段 from 表单 where 字段 between 某字段值 and 某字段值

  5. 空值检查

    select 字段 from 表单 where 字段 is null

数据过滤

  1. 组合 where 子句，使用多个 where 子句进行数据过滤，以 and 或者 or 子句的方式进行使用

  2. and 操作符：且

    select 字段 from 表单 where 字段1 = 某字段值 and 字段2 = 某字段值

  3. or 操作符：或

    select 字段 from 表单 where 字段1 = 某字段值 or 字段2 = 某字段值

  4. in 操作符：指定条件范围

    select 字段 from 表单 where 字段1 in (字段值1，字段值2)

  5. 计算次序：由于 where 可以包含任意数目的 and 或者 or，所以记得使用（）避免发生优先级相关错误

  6. not 操作符：否定它之后的所有条件

    select 字段 from 表单 where 字段1 not in (字段值1，字段值2)

使用通配符进行过滤

  1. like 操作符：表示使用通配符进行搜索

  2. %: 任何字符出现任意次数

    select 字段 from 表单 where 字段 like ‘aaa’

    匹配所有值等于 aaa 的字段

    select 字段 from 表单 where 字段 like ‘aaa%’

    匹配所有以 aaa 开头的字段

    select 字段 from 表单 where 字段 like ‘%aaa’

    匹配所有以 aaa 结尾的字段

    select 字段 from 表单 where 字段 like ‘%aaa%’

    匹配所有包含 aaa 的字段

  3. _: 匹配单个字符

ps: 能不用，就不用，性能不好

使用正则表达式进行搜索？

select 字段 from 表单 where 字段 regexp ‘aaa’

匹配所有值中包含 aaa 的字段

select 字段 from 表单 where 字段 regexp ‘aaa | bbb’

匹配所有值中包含 aaa 或包含 bbb 的字段

select 字段 from 表单 where 字段 regexp ‘[abc]’

匹配所有包含字符 a 或 b 或 c 的字段

select 字段 from 表单 where 字段 regexp ‘[ ^ abc]’

匹配所有除了字符 a 或 b 或 c 的字段
